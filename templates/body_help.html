Помощь

  • Для кого это приложение
    Данное приложение создано для самостоятельной практики студентов и для помощи преподавателям в подготовке методических материалов.

    Приложение ориентированно на обучение научному стилю речи русского языка, поэтому минимальный рекомендуемый уровень владения языком — В1.

  • Возможности приложения
    В данном приложении реализована возможность автоматической генерации заданий нескольких типов на основе текстов из корпуса или текста, введенного пользователем.

    После ввода критериев создания теста (см. пункт «Генерация заданий») загружается страница с заданиями. На ней содержится краткая инструкция по выполнению и сами задания. Каждому упражнению соответствует поле для введения ответа, слова в начальной форме для заполнения пропусков, поле для отметки упражнения как некорректного. Если отметить упражнение как содержащее ошибку, то появится поле для ввода комментария. Это позволит улучшить работу приложения.

    Внизу страницы расположена кнопка «Скачать файл с заданием», позволяющая загрузить файл в PDF-формате с тестом, и кнопка «Проверить» для перехода к странице с результатами.

    После выполнения проверки правильно решенные упражнения выделены зеленым, неправильно — красным, не оцениваемые — белым. Вверху странице указан общий процент верно решенных упражнений в тесте.

    В личном профиле (см. пункт «Личный профиль») можно увидеть статистику по всем решенным заданиям.

  • Генерация заданий
    На данной странице предлагается ввести следующие параметры:

    • источник текста («Корпус веб-приложения» или «Свой текст»);
    • тематику текста (доступно при работе с корпусом веб-приложения);
    • количество упражнений в тесте (в диапазоне от 1 до 99, по умолчанию 5);
    • показ грамматических подсказок.

    Обязательными для заполнения являются поля: «источник текста», «количество упражнений в тесте» и «показ грамматических подсказок».

    При выборе пункта «Свой текст» появляются поля для ввода текста и загрузки текстового файла. Для загрузки подходят только файлы txt-формата.
    *Если одновременно и ввести текст, и загрузить файл, что для генерации будет использован введенный текст.

    После формы расположена кнопка «Сгенерировать», которая после заполнения всех полей перенаправляет пользователя на страницу с заданиями.

  • Личный профиль
    Для создания профиля необходимо пройти краткую процедуру регистрации: указать адрес электронной почты, логин, ФИО и придумать пароль.

    После авторизации доступна страница со всеми решенными заданиями, сгруппированными в тесты, ответами к ним, оценками и датами выполнения. Также в профиле расположена диаграмма, демонстрирующая процентное соотношение верно и неверно решенных упражнений.
